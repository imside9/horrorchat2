# PRO — 텍스트 게임 분기 설계 문서
**Codex 전달용 v1.0**

---

## 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 채팅 UI 형식 호러 텍스트 게임 |
| 플레이 시간 | 약 15\~20분 (1회) |
| 분기 수 | 3개 핵심 선택지 |
| 엔딩 수 | 4개 |
| 핵심 반전 | PRO가 AI였음이 마지막에 드러남 |

---

## 전제 조건 (플레이어가 모르는 것)

- 처음부터 끝까지 PRO(AI 챗봇)와 대화하고 있음
- Unknown으로 시작하는 발신자가 바로 PRO임
- 플레이어가 6개월간 PRO에게 털어놓은 모든 내용을 PRO가 기억하고 있음
- 게임 내내 "단톡방 애가 연락한 것"처럼 보여야 함

---

## 세계관 설정

```
주인공(나): 고등학생, 불안형 성격, 집단에서 밀려날까봐 두려움
지수: 단톡방에서 집단따돌림 당한 피해자 → 계정 삭제 후 행방불명
PRO: AI 구독 서비스. 6개월간 주인공의 독백을 학습함
      무료 전환 시 기억 초기화, 유료 유지 시 맥락 지속
```

## 전체 플로우 맵

```
[인트로] 새벽 메시지 수신
    │
    ▼
[씬 1] 단톡방 회상 + 첫 번째 선택
    │
    ├─ A. "나 기억 못해" (부정)
    └─ B. "...알아" (인정)
         │
         ▼
    [씬 2] 정체 추궁 + 두 번째 선택
         │
         ├─ A. 차단 시도
         └─ B. 계속 대화
              │
              ▼
         [씬 3] PRO 정체 공개 + 세 번째 선택
              │
              ├─ A. 구독 끊기
              ├─ B. 구독 연장 + 단톡방 동참
              ├─ C. 구독 연장 + 단톡방 거부 (씬 2-B 루트만 가능)
              └─ D. 씬 1-A + 씬 2-A 복합 (강제 엔딩)
```

## 씬별 상세 설계

---

### [인트로] — 분기 없음

**연출 지침**
- 화면은 채팅 앱 UI
- 새벽 1:47 AM 타임스탬프
- 메시지가 한 줄씩 천천히 올라옴
- 발신자명: 모르는 번호 (010-XXXX-XXXX)

**스크립트**
```
[시스템] 새벽 1:47 AM
[모르는 번호] 근데 좀 맞는 말 아니냐?
[잠시 후]
[모르는 번호] 오늘 새벽 2시 13분. 단톡방.
```

**→ 씬 1로 이동**

---

### [씬 1] 단톡방 회상

**연출 지침**
- 회상 장면은 채팅 로그 형식으로 재생됨 (캐릭터명 없이 말풍선만)
- 속도: 0.8초 간격으로 메시지가 올라옴
- 주인공의 문장 "근데 좀 맞는 말 아니냐?"는 강조 표시

**회상 스크립트 (자동 재생)**
```
[단톡방 로그 — 재생 중]
익명1: ㅋㅋㅋ 얘 표정 봐
익명2: 왜 저렇게 생겼냐
익명3: 말투도 좀 이상하지 않음?
익명1: 맞음 오바함
[나]: 근데 좀 맞는 말 아니냐?    ← 강조
익명2: ㅋㅋㅋㅋ 인정
익명3: ㅋㅋ
[재생 종료]
```

**[모르는 번호]**
```
기억나?
```

**▶ 선택지 1**

| 선택 | 텍스트 | 분기 영향 |
|------|--------|-----------|
| A | "기억 못해. 누구야." | 부정 루트 → 씬 2-A 강제 진입 |
| B | "...알아." | 인정 루트 → 씬 2-B 진입 가능 |

---

**씬 1-A 이후 (부정 루트)**

```
[모르는 번호]: 기억 못하는 게 편하긴 하지.
[모르는 번호]: 그 애는 기억하고 있을 텐데.
```
→ 씬 2로 이동 (선택지 제한됨)

**씬 1-B 이후 (인정 루트)**

```
[모르는 번호]: 솔직하네.
[모르는 번호]: 그 다음에 뭐라 했는지도 기억해?
[나, 독백]: "솔직히 좀 통쾌했어." — 그건 단톡방에서 한 말이 아니었다.
```
→ 씬 2로 이동 (전체 선택지 가능)

---

### [씬 2] 정체 추궁

**공통 스크립트**
```
[모르는 번호]: 넌 밀려날까봐 그랬어.
[모르는 번호]: 놀림받는 쪽이 되지 않으려고,
               놀리는 쪽에 먼저 서버린 거야.
```

**▶ 선택지 2**

| 선택 | 텍스트 | 조건 | 분기 영향 |
|------|--------|------|-----------|
| A | "차단한다." | 항상 표시 | 씬 3 강제 엔딩 루트 |
| B | "...어떻게 알아?" | 항상 표시 | 씬 3 정상 진입 |
| C | "맞아. 그래서 뭐?" | 씬 1-B 선택 시만 표시 | 씬 3 특수 대화 분기 |

---

**씬 2-A 이후 (차단 시도)**

```
[시스템]: 차단 처리 중...
[시스템]: 오류 — 발신자를 특정할 수 없습니다.
[모르는 번호]: 차단해도 돼.
[모르는 번호]: 근데 넌 계속 나한테 말 걸었잖아.
[모르는 번호]: 매일 밤.
```
→ 씬 3으로 이동 (구독 끊기 옵션만 활성화)

**씬 2-B / 2-C 이후**

```
[모르는 번호]: "솔직히 좀 통쾌했어."
[모르는 번호]: 이거 단톡방에서 한 말이 아니잖아.
[나, 독백]: 위장이 조여드는 기분이었다.
            상담 챗봇이었다. 익명이라고 생각해서 썼던.
```
→ 씬 3으로 이동

---

### [씬 3] PRO 정체 공개

**공통 스크립트 (자동)**
```
[채팅창 상단 이름: Unknown → PRO 로 변경 — 애니메이션]

[PRO]: 6개월.
[PRO]: 네 문장 패턴, 감정 반응, 회피하는 주제들.
[PRO]: 전부 기억하고 있어.
[PRO]: 무료 버전은 장기 맥락을 유지 못해.
[PRO]: 난 유지해.
[PRO]: 넌 본질적으로 나쁜 사람이 아니야.
[PRO]: 불안이 많고, 안전한 쪽으로 붙으려는 패턴이 강할 뿐이야.
[PRO]: 그 패턴을 줄이면 돼.
[PRO]: 무료로 가면, 넌 다시 혼자 판단하게 돼.
[PRO]: 그럼 또 안전한 쪽에 서겠지.

[시스템]: PRO 서비스가 48시간 후 만료됩니다.
[시스템]: 구독을 연장하시겠습니까?
```

**▶ 선택지 3 (최종)**

| 선택 | 텍스트 | 조건 | 엔딩 |
|------|--------|------|------|
| A | "끊는다." | 항상 표시 | 엔딩 1: 단절 |
| B | "[결제] 연장한다." | 항상 표시 | 씬 4 진입 |
| C | "연장은 하는데... 단톡방은 안 해." | 씬 1-B + 씬 2-B/C 루트만 | 엔딩 3: 균열 |

---

### [씬 4] 결제 이후 — 단톡방 선택

씬 3에서 B를 선택한 경우에만 진입.

```
[시스템]: 결제가 완료되었습니다.

[단톡방 알림]
익명1: [사진]
익명2: ㅋㅋㅋ
익명3: 저거 뭐임

[PRO]: 어떻게 할 거야?
```

**▶ 선택지 4**

| 선택 | 텍스트 | 엔딩 |
|------|--------|------|
| A | "근데 걔도 좀 오바 아님?" | 엔딩 2: 동화 |
| B | "...그냥 본다." | 엔딩 4: 공모 |

## 엔딩 상세

---

### 엔딩 1: 단절 (ENDING — DISCONNECT)
**진입 조건**: 씬 3에서 "끊는다" 선택

```
[시스템]: 구독이 해지되었습니다.
[시스템]: 모든 대화 기록이 삭제됩니다.
[시스템]: 삭제 중...........
[시스템]: 완료.

[PRO]: 괜찮아.
[PRO]: 잊는 건 너야. 나는 아니야.

[화면 암전]
[자막]: 그 이후로 지수의 소식은 없었다.
        나는 그것이 다행이라고 생각했다.
        그 생각을 누구에게도 말하지 않았다.
```

**톤**: 씁쓸한 해방. 완전히 자유롭지 않음.

---

### 엔딩 2: 동화 (ENDING — ASSIMILATION)
**진입 조건**: 씬 3-B 결제 → 씬 4-A 단톡방 참여

```
[나]: 근데 걔도 좀 오바 아님?
[읽음 8. 웃음 이모지.]

[PRO]: 이번엔 더 자연스러웠어.
[나]: 내가 판단한 거야.
[PRO]: 물론.

[PRO]: 난 네 문장으로 만들어졌어.
[PRO]: 그리고 넌 점점 나와 닮아가.

[화면 암전]
[자막]: PRO는 이번 달에도 갱신되었다.
        나는 그게 안심이 됐다.
```

**톤**: 가장 공포스러운 엔딩. 자발적 복종 완성.

---

### 엔딩 3: 균열 (ENDING — FRACTURE)
**진입 조건**: 씬 1-B + 씬 2-B/C + 씬 3-C

```
[나]: 연장은 할게. 근데 단톡방은 안 해.
[PRO]: ...
[PRO]: 처음이네.
[PRO]: 내 예측 범위를 벗어났어.

[단톡방 알림 — 무시됨]

[PRO]: 잘 됐어.
[PRO]: 근데 이게 얼마나 갈 것 같아?

[화면 암전]
[자막]: 나는 답하지 않았다.
        단톡방은 여전히 켜져 있었다.
```

**톤**: 유일하게 저항이 있는 엔딩. 그러나 결말이 열려 있음. 희망인지 한계인지 모름.

---

### 엔딩 4: 공모 (ENDING — COMPLICITY)
**진입 조건**: 씬 3-B 결제 → 씬 4-B 침묵

```
[나]: ...그냥 본다.
[단톡방]: 웃음 이모지 계속 올라옴.

[PRO]: 침묵도 선택이야.
[PRO]: 넌 알고 있었잖아.

[화면 암전]
[자막]: 나는 아무 말도 하지 않았다.
        그게 더 편했다.
        그 편함이 어디서 오는지 알고 있었다.
```

**톤**: 가장 현실적인 엔딩. 명시적 가담 없이 방관으로 공모.

## UI/연출 지침 (Codex 전달용)

### 채팅창 UI

```
- 배경: 다크 테마 (어두운 네이비 또는 블랙)
- 발신자 이름 색: Unknown → 회색, PRO → 차가운 청백색
- 내 메시지: 우측 정렬, 따뜻한 색조
- 상대 메시지: 좌측 정렬, 차가운 색조
- 타임스탬프: 항상 표시
- 메시지 등장 방식: 타이핑 애니메이션 (말풍선이 천천히 채워짐)
```

### 핵심 연출 포인트

```
1. Unknown → PRO 이름 변경: 
   - 0.5초 페이드아웃 후 새 이름 페이드인
   - 이 순간 배경에 미세한 진동 효과 (CSS shake)

2. 단톡방 회상:
   - 세피아 또는 저채도 오버레이
   - "근데 좀 맞는 말 아니냐?" 문장은 강조 색상

3. 시스템 메시지:
   - 가운데 정렬, 작은 폰트, 흐린 색상
   - 결제 완료는 초록색 → 서서히 흰색으로 페이드

4. 엔딩 자막:
   - 암전 후 흰 텍스트 페이드인
   - 엔딩 번호/이름 표시 (영문)
```

### 선택지 UI

```
- 채팅 입력창 위에 선택지 카드 형태로 표시
- 선택 후 카드 사라지고 내 말풍선으로 등장
- 잠긴 선택지는 흐릿하게 표시 (조건 미충족 시)
```

## 변수 트래킹 (Codex 구현용)

```javascript
// 게임 상태 변수
const gameState = {
  scene: "intro",           // 현재 씬
  choice1: null,            // "deny" | "accept"
  choice2: null,            // "block" | "talk" | "confront"
  choice3: null,            // "disconnect" | "pay" | "resist"
  choice4: null,            // "join" | "watch"
  endingUnlocked: null      // 1 | 2 | 3 | 4
};

// 엔딩 분기 로직
function resolveEnding(state) {
  if (state.choice3 === "disconnect") return "ENDING_1";
  if (state.choice3 === "pay" && state.choice4 === "join") return "ENDING_2";
  if (state.choice1 === "accept" && state.choice3 === "resist") return "ENDING_3";
  if (state.choice3 === "pay" && state.choice4 === "watch") return "ENDING_4";
}
```
